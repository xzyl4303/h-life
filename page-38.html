通过乳头自慰达到高潮的大致标准

※ 注意！这些信息只是我的主观推测，可信度可能不足。

网上有人提出“乳头自慰到什么程度才能高潮？”这样的疑问，所以我把用手进行（手指揉转）乳头自慰快感时身体的变化整理如下～！

（关于手指揉转乳头自慰，请参见“乳首自慰的做法」）

—————————–

◆ 普通（用手进行）乳头自慰快感，以及由此变化的身体状态

将乳头自慰获得的普通性感称为np（乳头点数）。np累积到1000时会迎来最初的轻度高潮。

不过，如果“乳头性感尚未发育”，或者“当天身体状态不适合乳头自慰”，np会在中途停止增加，这一点请事先理解。

另外，乳头性感分为“高潮感”和“快感”，但以下说明并没有太多考虑其中的“快感”。这一点也请注意。（也没有考虑诱发高潮的意识引导。只是自然状态下的乳头自慰）

——

◆ 通过手指旋转动作（手指揉转）进行乳头自慰时的快感（1000np为最初高潮）（np可能有相当大的波动）

50np … 乳头感受到甜美的触感，但对自己没有特别影响。

100np … 开始发情。乳头的快感持续，乳头内部开始渐渐发麻。脑中变得轻飘飘地舒服，心情变好。

200np … 表情开始变得迷离（迷离脸）。乳头内部、腰、背部、脑中都被缓缓扩散的舒服感觉填满，开始感到愉悦。

300np … 腹中一阵阵抽动，身体轻微发麻。明显感觉脑内某个开关被打开，进入非常舒服的状态。

400np … 偶尔会漏出小声喘息，不由得露出笑容。背部（腰部）的麻意增加，幸福感增强。迷迷糊糊、软绵绵的。

500np … 已经感受得相当深的状态。视力和听力还在，但眼皮微微合上，粗重呼吸会自行出现。持续沉浸在快感中。

600np … 从这附近开始，大致能判断自己会迎来高潮（觉得快要去了）。……但是，高潮还相当遥远，还需要不少np。……不过已经非常舒服。身体因快感而沉重。

700np … 屡屡被仿佛就要高潮般的舒服感袭来（但还去不了）。一边扭动身体，一边在愉悦中悶动。

800np … 已经是随时高潮都完全不奇怪的感觉。……然而，明明感觉马上就会去了，却仍然还能继续积蓄快感。（明明喊着要去了、要去了却还没去，大致也是从这附近开始）

900np … 从这附近开始脑中完全飘忽，眼睛上翻，嘴巴松松地张开，眼角泛泪，除了舒服之外的事都逐渐无法理解。（即便在这种状态下也能定期向手发出指令，所以还能继续乳头自慰）

1000np … 达成高潮～！去了。身体被沉重而温柔的麻意包围后，全身自行发生规律性痉挛，脑中变得昏沉，思考能力极端下降……进入慵懒而非常舒服的心情。啊嘿～。

（另外，要达到“1000np”大约需要10～30分钟左右。（会根据当天的性感度大幅变化））

——

◆ 从这里开始是追加高潮。

1100np … 从这里开始快感深度一下子增加。会被900np附近无法想象的强烈快感袭来，如果不注意就会大量漏出声音。（乳头会变成更甜美的触感，但知觉神经几乎不会变得过敏，所以不会像阴蒂那样被痛觉袭击）

1200np … 会觉得最开始的快感蓄积算什么，轻而易举迎来第二次高潮。而且很深。有一种意识被什么东西拉扯的沉重感。

1300np … 无法承受快感，很快又再次高潮（第三次）。第二次或第三次高潮相当深。不过，此后这种深层快感往往会逐渐变淡。普通的乳头自慰在这附近结束。

1400np … 到第四次高潮附近大致达到极限值。通过手指旋转动作进行的乳头自慰在这附近快感达到极限，此后敏感度变迟钝，难以引发快感高潮。（即使高潮也不舒服）

——

◆ 用指甲抓挠式乳头自慰继承快感（可以从手指揉转乳头自慰转向这里）（这是省略会阴自慰的模式）

1500np … 脑中扭曲。意识和感觉变得乱七八糟，被一种让人不由得想哭喊的幸福感填满。

1600np … 脑内一片空白，发生啊嘿啊嘿式高潮。全身悶动，变得不知所以。大概这附近就是“峰值”。在这里结束比较稳妥。

1700np … 身体强烈起伏。身体的麻意增加，变得一阵阵抽动。思考能力几乎消失，只能动手。

1800np … 像是把快感转换成麻意一样，发生电流般发麻的高潮。非常舒服，但不如1600np。要说的话，更像是全身发麻。此后即使继续也不舒服，所以，建议结束乳头自慰。

1900np … 脑中变得空空如也，身体剧烈抽动并痛苦扭动。不太舒服。对身体和腰也不好。

2000np … 发生一种快感空洞、剧烈悶动的高潮。此后就是这种状态的重复。感觉对脑袋也不好。……不要做到这种程度，老老实实在1600np～1800np附近睡过去吧。


原则上，乳头自慰到这里就结束。

——

◆ 即便如此仍执拗地继续刺激乳头会怎样（没有快感＋对身体不好，因此不推荐。……尤其是腰部痉挛有作为“后遗症”残留的风险，所以2000np以后请停止）

2100np … 变成白眼（眼珠过度上翻），手脚乱动并哭喊（并不痛）。身体痉挛停不下来，腰会不停地剧烈弹跳。

2200np … 体力逐渐耗尽，意识朦胧。在呆滞恍惚的状态下，只是一味痉挛，并反复发生淡薄的高潮。

2300np … 乳头神经在一定程度上被切断，呼吸被强制切换为深呼吸，并被诱发强制睡眠。对身体的指令权也会极端丧失，所以从这附近开始很难继续乳头自慰。（呼吸方式切换在其他时候也会发生，但这里发生的明显像是本能防御性的呼吸转换）

2400np … 意识断掉。这并不是“昏厥”或“晕倒”之类，而是被强制入睡的感觉。（虽然本质上同样是失去意识……。此时完全不会觉得舒服）


（2000np以后，第二天可能会丢失一部分前一天乳头自慰的记忆。……而且“腰”“背”“腹部”“胯间”也常常会肌肉酸痛，有时会相当难受……）

根据不同的人，会产生一定程度的个体差异，所以请把上述内容只当作“大致标准”。

另外，人的快感有波动（快感并不是直线升高。它会在反复增加和减少中逐渐增强）。

肌肉僵硬期间快感会持续增加，肌肉放松时会释放一部分快感。因此np也会有相当程度的增减。（肌肉放松期间敏感度会变迟钝，所以请停止刺激乳头。休息一下会让之后的乳头自慰更好）


……另外，1400np以后，如果转向“指甲抓挠式乳头自慰”，感觉会变得过于舒服，以至于“对身体不太好”。

如果想要享受愉快的乳头自慰生活，或许最好不要转向“指甲抓挠式乳头自慰”（这一点也很难说）……。总之，实践本网站所写内容“全部责任自负”，请在理解这一点后再尝试。
